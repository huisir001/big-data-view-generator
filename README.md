# 大数据可视化平台快速生成器
> 拖拽式大屏页面快速生成平台

## 当前支持组件
- 柱状图
- 折线图
- 双Y轴图
- 饼状图
- 仪表盘
- 雷达图
- 散点图
- 默认地图（山西地图/中国地图）
- 进度图（6种）
- 边框组件（13种）

## 注意

> 暂无

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# test with hot reload at localhost:8080
npm run test

# pro with hot reload at localhost:8080
npm run pro

# build for production with minification
npm run build
```
